# Implementation Plan - Automatic Item Property Population

The goal is to automatically fill default values for item attributes when an item is selected in the "Add Item" form. This will include logic to pick the maximum value in case of a range (e.g., 20%-30% ED becomes 30%).

## User Review Required

> [!IMPORTANT]
> The auto-population will use a predefined mapping from Diablo II internal property names (like `ac%`, `dmg%`) to the form fields (like `enhanced_defense`, `enhanced_damage`). If any mapping is incorrect or missing, it will need further refinement.
